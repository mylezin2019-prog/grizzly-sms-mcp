# Grizzly SMS Skill for OpenClaw

This skill enables OpenClaw to use the Grizzly SMS MCP server for SMS verification and virtual phone numbers.

## What it does

- Request virtual numbers for 500+ services (Telegram, WhatsApp, etc.)
- Get SMS verification codes
- Check balance and get crypto wallet for top-up
- Manage activations (complete, cancel, request new code)

## Installation

1. **Install this skill** (instructions for agent):
   ```
   clawhub install GrizzlySMS-Git/grizzly-sms
   ```

2. **Install and configure the MCP server** — see [CONFIG.md](CONFIG.md)

3. **Restart OpenClaw** to load the MCP server
